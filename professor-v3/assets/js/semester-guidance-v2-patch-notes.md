# 학기별 논문지도 V2 패치 노트

## 버전 2.1.0 (2025-12-02)

### 🎨 UI 대폭 간소화

#### 1. 학생 정보 영역 개선
**변경사항:**
- 지도교수 정보를 학생 정보 카드로 이동
- 4열 그리드 레이아웃으로 변경 (전공, 학위과정, 현재 단계, 지도교수)

**목적:**
- 정보 통합으로 화면 공간 효율성 증대
- 학생 관련 모든 정보를 한눈에 파악

#### 2. 통계 영역 제거
**삭제된 항목:**
- 지도교수 (→ 학생 정보로 이동)
- 등록된 주차 (N/15주차)
- 실적 입력 완료 (N주차)

**이유:**
- 불필요한 중복 정보 제거
- 각 주차 카드에서 이미 실적 개수 표시
- 화면 집중도 향상

#### 3. 아코디언 기능 제거
**변경사항:**
- 주차 카드 헤더 클릭 기능 제거
- "전체 펼치기/전체 접기" 버튼 제거
- 모든 카드 항상 펼쳐진 상태로 표시
- 아코디언 토글 함수 제거 (`toggleWeekCardV2`, `toggleAllWeeksV2`)

**새로운 구조:**
```
┌─────────────────────────────┐
│ N주차 - 주제 | 방식 | 실적 N건 │  ← 제목만 표시 (클릭 불가)
├─────────────────────────────┤
│ 계획 내용 또는 입력 폼       │
│                              │
│ 실적 및 교수의견             │
│ - 실적 1                     │
│ - 실적 2                     │
│ - 입력 폼                    │
└─────────────────────────────┘
```

**목적:**
- 연속적인 입력 작업에 최적화
- 클릭 동작 제거로 사용성 개선
- 15주차 전체를 한 화면에서 빠르게 스크롤 가능

#### 4. 아이콘 제거
**제거된 아이콘:**
- "실적 및 교수의견" 헤더의 댓글 아이콘
- "수정" 버튼의 편집 아이콘
- "삭제" 버튼의 휴지통 아이콘
- 아코디언 화살표 아이콘

**이유:**
- 텍스트만으로 충분히 명확한 의미 전달
- 시각적 노이즈 감소

#### 5. CSS 최적화
**삭제된 스타일:**
- `.week-card-header` 관련 스타일
- `.open` 클래스 관련 스타일
- `fadeIn` 애니메이션
- 아코디언 토글 transition

**결과:**
- CSS 파일 크기 약 60% 감소
- 불필요한 DOM 조작 제거

## 버전 2.0.1 (2025-12-02)

### 🐛 버그 수정

#### 1. `openSemesterPlanEditModal` 미구현 에러 수정
**문제:**
```javascript
Uncaught ReferenceError: openSemesterPlanEditModal is not defined
```

**원인:**
- V2 코드에서 `openSemesterPlanEditModal` 함수가 export되었으나 실제 함수는 구현되지 않음
- 라인 122에서 "계획 수정" 버튼 클릭 시 호출되는 함수

**해결:**
- `openSemesterPlanEditModal()` 함수 구현 완료
- `proceedToWeeklyEdit()` 함수 추가 (수정 전용)
- 기존 계획의 학기 시작일을 불러와서 표시
- V1의 15주차 수정 화면으로 연결

**관련 코드:**
```javascript
// 새로 추가된 함수들
function openSemesterPlanEditModal() { ... }
function proceedToWeeklyEdit() { ... }

// Export에 추가
window.openSemesterPlanEditModal = openSemesterPlanEditModal;
window.proceedToWeeklyEdit = proceedToWeeklyEdit;
```

## 버전 2.0.0 (2025-12-02)

### ✨ 새로운 기능

#### 1. 댓글 방식 실적 입력
- SNS 스타일 인라인 입력 폼
- 모달 팝업 제거
- 연속 입력 가능

#### 2. 학기 선택 드롭다운
- 최근 5년간 학년도/학기 조회
- 계획 존재 여부 표시 (✓)
- 과거 학기 실적 조회 및 추가

#### 3. 통합 화면
- 등록 + 조회를 단일 화면으로
- 계획 없음 → "15주차 계획 등록" 버튼
- 계획 있음 → 주차별 카드 + 실적 입력폼

#### 4. 개선된 UX
- 교수 아바타 (원형 뱃지)
- 실적 개수 뱃지 (💬 N건)
- 주차 카드 아코디언
- 전체 펼치기/접기 기능
- 자신의 실적만 수정/삭제
