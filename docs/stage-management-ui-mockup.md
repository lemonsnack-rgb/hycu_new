## 1. 워크플로우 목록 화면

```
┌──────────────────────────────────────────────────────────────────────────┐
│ 논문지도 워크플로우 관리                        [+ 워크플로우 추가] │

│ 번호 │ 워크플로우명      │ 학위   │ 단계 수 │ 단계 구성                │ 관리   │

│  1   │ 석사 표준 과정    │ [석사] │ 6단계   │ 1.연구계획서제출 →       │ 수정   │
```

## 4. 사용자 상호작용 시나리오 (애니메이션)

### 4.1 "단계 추가" 버튼 클릭

**Before:**
```
┌────────────────────────────────────────────┐
│      [+ 단계 추가]                          │
├────────────────────────────────────────────┤
│                                            │
│    추가된 단계가 없습니다.                  │
│                                            │
└────────────────────────────────────────────┘
```

**After (새 단계 카드 생성):**
```
┌────────────────────────────────────────────┐
│      [+ 단계 추가]                          │
├────────────────────────────────────────────┤
│ ┌────────────────────────────────────────┐ │
│ │ 단계 1               [삭제] [↑] [↓]    │ │
│ ├────────────────────────────────────────┤ │
│ │ 카테고리                                │ │
│ │ [연구계획서 제출 ▼] ← 기본 선택        │ │
│ │                                        │ │
│ │ 단계 이름                               │ │
│ │ [_____________________] ← 빈칸         │ │
│ │                                        │ │
│ │ ... (나머지 입력 필드)                  │ │
│ └────────────────────────────────────────┘ │
└────────────────────────────────────────────┘
```

### 4.2 평가표 선택 시 심사 기간 활성화

**평가 없음 선택 시:**
```
┌────────────────────────────────────────────┐
│ 평가표                                      │
│ [평가 없음 ▼]                               │
│                                            │
│ 심사 시작일      심사 마감일                │
│ [___________]   [___________]  ← 회색      │
│ (비활성화)      (비활성화)                  │

```

**평가표 선택 시:**
```
┌────────────────────────────────────────────┐
│ 평가표                                      │
│ [예비 심사 평가표 ▼]                        │

│ [2025-05-01]    [2025-05-15]  ← 흰색      │
│ (활성화)        (활성화)                    │
└────────────────────────────────────────────┘
```

### 4.3 단계 순서 변경

**Before:**
```
┌──────────────────────────┐
│ 단계 1: 연구계획서 제출   │  [↑비활성] [↓]
├──────────────────────────┤
│ 단계 2: 1차 예비심사      │  [↑] [↓]
├──────────────────────────┤
│ 단계 3: 2차 예비심사      │  [↑] [↓비활성]
└──────────────────────────┘
```

**단계 2에서 ↑ 클릭 후:**
```
┌──────────────────────────┐
│ 단계 1: 1차 예비심사      │  [↑비활성] [↓]  ← 순서 변경됨
├──────────────────────────┤
│ 단계 2: 연구계획서 제출   │  [↑] [↓]
├──────────────────────────┤
│ 단계 3: 2차 예비심사      │  [↑] [↓비활성]
└──────────────────────────┘
```
