## 주요 요구사항 카테고리

1. **로그인 및 사용자 동기화**
   - ERP 연동 사용자 동기화
   - 배치 스케줄러 구현
   - 접근 제어 정책 적용

2. **권한 및 부서 관리**
   - 부서별/신분별 권한 설정
   - 대리 로그인 기능
   - 메뉴 동적 제어

3. **논문 지도 교수 배정**
   - 주지도교수/공동지도교수 배정
   - 교수 검색 모달
   - 소속 학과 우선 정렬

4. **논문 지도 단계(Workflow) 관리**
   - 학과별 워크플로우 설정
   - 학생별 워크플로우 부여
   - 단계 이관 처리

5. **심사 및 평가 관리**
   - 심사위원 배정
   - 점수형/P/F형 평가표
   - 자동 판정 로직 (2/3 이상 통과)

6. **일정 관리**
   - 본부 공통 일정 관리
   - 학기별 일정 설정
   - 제출 가능 기간 안내

## 프로토타입 개발 기간

- **Phase 1**: 핵심 화면 구현 (3주)
  - 교수 배정, 학생 단계 관리, 심사위원 배정, 심사 점수 입력, 안내 화면
- **Phase 2**: 권한 및 심사 관리 (2주)
  - 권한 관리 UI, 대리 로그인, 심사 결과 판정, 메뉴 동적 제어, 일정 조회
- **Phase 3**: UI/UX 개선 및 최적화 (1주)
  - 평가표 구성, 워크플로우 설정, 테이블 UI 개선, 로딩 상태 표시
- **총 예상 기간**: **6주 (약 1.5개월)**
