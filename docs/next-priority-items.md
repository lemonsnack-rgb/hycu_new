### P1 (필수 - 1차 구현) - 4개 항목

#### **[P1-1] 관리자 - 지도 학생 관리 (통합 메뉴)** ⭐ **최우선**
**예상 소요**: 5일

**메뉴 위치**: 논문지도 > 지도 학생 관리

**기능**:
1. **지도교수 배정**
   - 연구계획서 제출 학생 목록 테이블
   - 연구계획서 보기 모달
   - 주지도교수 검색 모달 (라디오 버튼, 1명)
   - 공동지도교수 검색 모달 (체크박스, N명)
   - 소속 학과 우선 표시, 타과 포함 체크박스
   - 전임교원 우선 정렬 (배지 표시)
   - 배정 처리 로직

2. **학생 지도 단계 변경 (이관)**
   - 학생별 현재 지도 단계 표시
   - 다음 단계로 이관 버튼
   - 조건 체크 로직 (심사 완료 여부 등)
   - 조건 미충족 시 경고 모달

**신규 파일**:
- `admin/admin-student-management.html`
- `admin/assets/js/student-management.js`

**Mock 데이터 필요**:
- mockResearchProposals (연구계획서 20건)
- mockAdvisorAssignments (교수 배정 데이터)
- mockStudentStages (학생별 지도 단계 데이터)

**IA 문서 근거**: 행 21-23
- 비고: "논문 지도 학생 목록을 구성(배치 등)하는 기능. 학생의 목록을 조회하고, 관리자가 학생별로 해당 학생의 지도 단계를 변경(이관)하는 화면. 학생별 지도교수를 할당(배정)하는 기능"
