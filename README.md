## 🚀 사용 방법

### 1. 파일 배포
```
1. 루트에 common-style.css, common-utils.js 배치
2. admin/, professor/, student/ 폴더를 각각 배포
```

### 2. 경로 확인
각 HTML 파일에서 공통 파일 경로:
```html
<link rel="stylesheet" href="../common-style.css">
<script src="../common-utils.js"></script>
```

- ✅ 기존 JavaScript 로직 100% 유지
- ✅ 데이터 흐름 정상
