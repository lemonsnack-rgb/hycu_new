## ⚡ 3분 테스트

### **Step 1: 파일 열기 (10초)**
```
1. ZIP 압축 해제
2. index.html을 브라우저에서 열기
```

### **Step 2: 심사 목록 확인 (30초)**
```
1. [심사관리] 메뉴 클릭
2. 심사 목록 확인:
   
   No │ 학생    │ 단계      │ 논문제목           │ 내 역할    │ 진행 │ 마감일 │ 상태
   ─────────────────────────────────────────────────────────────────────────
   1  │ 김철수  │ 연구계획  │ 딥러닝 기반...    │ 🔹 위원   │ 1/3  │ 11/20  │🟡진행중
   2  │ 이영희  │ 중간논문  │ 블록체인 기반...  │ ⭐ 위원장 │ 3/3  │ 11/25  │🔵심사완료
   3  │ 박민수  │ 최종논문  │ AI 기반...        │ 🔹 위원   │ 4/4  │ 11/15  │🟢승인

3. 필터 테스트:
   - 단계: [중간논문] 선택 → 1건만 표시
   - [전체] 복원
```

### **Step 3: 심사위원 평가 (1분)**
```
1. "김철수 - 연구계획서" 클릭 (🔹 심사위원)

2. 평가표 작성:
   - 각 항목 슬라이더 조절
   - 실시간으로 총점 계산 확인
   - 예상 결과 표시 확인

3. 총평 작성:
   - 장점 입력
   - 개선점 입력
   - 종합 의견 입력

4. [💾 임시저장] 클릭
   → 토스트: "저장되었습니다"

5. [✅ 제출] 클릭
   → 확인 모달 → "제출 후 수정 불가" 경고
   → [확인]
   → 제출 완료 화면으로 전환
```

### **Step 4: 심사위원장 종합 평가 (1분)**
```
1. 목록으로 돌아가기
2. "이영희 - 중간논문" 클릭 (⭐ 심사위원장)

3. 하단 스크롤:
   → "⭐ 종합 평가 (심사위원장 전용)" 섹션 확인

4. 평가 현황:
   ✅ 김교수 (위원장) - 11/15 14:30
   ✅ 이교수 (위원) - 11/16 16:20
   ✅ 한교수 (위원) - 11/14 18:45

5. 점수 비교표 확인:
   - 항목별 점수
   - 심사위원별 점수
   - 평균 점수

6. 시스템 판정:
   - 평균: 83.3점
   - 기준: 75점
   - 판정: ✅ 합격

7. 최종 결정:
   - [승인] 선택
   - 의견: "모든 평가 우수. 승인"
   - [✅ 최종 결정 제출]

8. 목록 → 상태: 🟢 승인 확인
```

## 💡 TIP

### 점수 입력
```
💡 슬라이더를 드래그하면 실시간으로 총점이 계산됩니다.
💡 합격 기준과 비교하여 예상 결과가 표시됩니다.
💡 0.5점 단위로 입력 가능합니다.
```

### 임시저장
```
💡 작성 중인 평가는 언제든 임시저장할 수 있습니다.
💡 임시저장된 내용은 다음에 접속해도 유지됩니다.
💡 제출 전까지는 수정 가능합니다.
```

### 제출
```
⚠️ 제출 후에는 수정할 수 없습니다.
⚠️ 제출 전에 반드시 확인하세요.
⚠️ 확인 모달에서 취소 가능합니다.
```

### 심사위원장
```
⭐ 모든 심사위원이 제출 완료해야 종합 평가가 표시됩니다.
⭐ 최종 결정은 심사위원장만 할 수 있습니다.
⭐ 의견 작성은 필수입니다.
```
