## 🎨 디자인 특징

### 1. **완성형 ERP 스타일**
- **색상 팔레트 단순화**:
  - Primary: `#009DE8` (한양사이버대 블루)
  - Sidebar: `#1e293b` (다크 네이비)
  - Success: `#10b981` (녹색 - 완료)
  - Danger: `#dc2626` (빨강 - 긴급)
  - Info: `#3b82f6` (파랑 - 정보)
  - 중립 회색 계열만 사용
- **폰트**: Pretendard (Google Fonts)
- **디자인**: 각진 모서리, 조밀한 정보 배치, 전문적인 느낌

### 2. **이모지 제거, 플랫 아이콘만 사용**
- 모든 이모지 제거 (공지사항 아이콘 제외)
- 플랫 디자인 문자 아이콘 사용: `▪`, `▶`, `▌`, `▸`, `━`
- 상태 표시: 색상 배경 + 텍스트 뱃지

### 3. **사이드바 접기/펼치기**
- PC/모바일 모두 토글 가능
- 우측 하단 토글 버튼
- 접혔을 때: 60px (아이콘만)
- 펼쳤을 때: 240px (전체 메뉴)
- 부드러운 애니메이션 효과

## 🔧 주요 기능

### 1. **사이드바 토글**
```javascript
function toggleSidebar() {
    document.getElementById('sidebar').classList.toggle('collapsed');
}
```
- PC/모바일 모두 동작
- 부드러운 애니메이션

### 2. **학위 과정 필터**
```javascript
function changeDegreeFilter(filterId) {
    currentDegreeFilter = filterId;
    renderDegreeFilter();
    renderProcessMap();
    renderStudentCards();
    hideQuickView();
}
```
- 전체/석사/박사/통합 필터링
- 프로세스 맵과 학생 카드에만 적용
- 심사관리/피드백/일정은 필터 무관

### 3. **프로세스 맵 퀵뷰**
```javascript
// PC: 마우스 호버
onmouseenter="showQuickView('proposal')"
onmouseleave="hideQuickView()"

// Mobile: 클릭
onclick="showQuickView('proposal')"
```
- 해당 단계 학생 명단 테이블 표시
- 노란색 배경으로 강조

### 4. **심사 단계 탭**
```javascript
function changeReviewTab(tabId) {
    currentReviewTab = tabId;
    renderReviewTabs();
    renderReviewContent();
}
```
- 논문작성계획서/예비심사/본심사 탭
- 각 단계별 심사 대기/완료 분리

### 5. **네비게이션**
- `navigateToStudent(studentId)` - 학생 상세 화면
- `navigateToReview(reviewId)` - 심사하기
- `viewReviewResult(reviewId)` - 심사 결과 보기
- `navigateToFeedback()` - 온라인 피드백 화면
- `navigateToMeetingApproval()` - 미팅 승인 관리
