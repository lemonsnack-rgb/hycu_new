## ⏱️ 30초 테스트

### **Step 1: 열기 (5초)**
```
1. ZIP 압축 해제
2. index.html 더블클릭
3. [심사관리] 클릭
```

### **Step 2: 목록 확인 (10초)**
```
✅ 확인사항:
   - 우측 상단: "총 10건"
   - 테이블: 10개 행
   - 배지: 🟡🔵🟢🟠🔴
   - 역할: ⭐4개, 🔹6개
```

### **Step 3: 필터 테스트 (10초)**
```
1. 역할: [심사위원장] 선택
   → "4건 (전체 10건)" 표시
   → ⭐ 배지만 보임

2. [초기화] 클릭
   → "총 10건" 복원
```

### **Step 4: 상세 확인 (5초)**
```
1. "이영희 - 중간논문" 클릭
2. 스크롤 다운
3. "⭐ 종합 평가" 섹션 확인
   → 점수 비교표
   → 평균 83.3점
   → 시스템 판정: 합격
```

## 🔍 문제 해결

### Q1: 목록이 안 보여요
```
→ 브라우저 콘솔 확인 (F12)
→ review-data.js 로드 확인
→ CURRENT_USER 확인
```

### Q2: 빈 목록이 나와요
```
→ 현재 사용자: P003 (김교수)
→ P003이 모든 심사에 포함됨
→ 10건이 표시되어야 함
```

### Q3: 필터가 안 돼요
```
→ 검색 버튼 클릭 필요
→ 또는 Enter 키
→ 초기화 버튼으로 리셋
```
