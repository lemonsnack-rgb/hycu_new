# 대시보드 V2 - 시안4 세련된 데이터 중심형

## 📁 파일 구조

```
dashboard-v2/
├── index.html                     # 메인 HTML 파일
├── README.md                      # 본 문서
└── assets/
    ├── css/
    │   └── dashboard-v2.css      # 시안4 스타일 (이모지 제거)
    └── js/
        ├── dashboard-v2.js       # 대시보드 로직
        └── mock-data.js          # Mock 데이터 서비스
```

## 🎨 디자인 특징

### 1. **시안4 스타일 적용**
- **색상**:
  - Primary: `#009DE8` (한양사이버대 공식 파란색)
  - Sidebar: `#1e293b` (다크 네이비)
  - Background: `#f8f9fb` (밝은 회색)
- **폰트**: Pretendard (Google Fonts)
- **디자인**: 깔끔한 카드 기반, 데이터 중심 레이아웃

### 2. **이모지/아이콘 완전 제거**
- 모든 이모지를 텍스트 + 색상 배지로 대체
- 상태 표시: 색상 배경 텍스트 뱃지 사용
- 미팅 구분: `[온라인]` / `[오프라인]` 텍스트 배지
- 화살표: `→` 또는 `›` 문자 사용

### 3. **사이드바 접기/펼치기**
- 우측 하단 토글 버튼으로 제어
- 접혔을 때: 60px (아이콘만)
- 펼쳤을 때: 260px (전체 메뉴)
- 부드러운 애니메이션 효과

## 📊 화면 구성

### **Zone A: 프로세스 맵 (Interactive Navigator)**
```
┌────────────────────────────────────────────┐
│ 학위 과정 필터                              │
│ [전체] [석사] [박사] [통합]                 │
│                                            │
│ 프로세스 단계                               │
│ [논문작성계획서 2] → [예비심사 1] →         │
│ [본심사 3] → [최종제출 4]                   │
│                                            │
│ 퀵 뷰 패널 (호버/클릭 시 표시)              │
│ - [석사] 김철수 (4학기) | 보완요망 | [상세] │
└────────────────────────────────────────────┘
```

**기능:**
- 학위 과정별 필터링 (전체/석사/박사/통합)
- 단계별 학생 수 표시
- 호버/클릭 시 해당 단계 학생 명단 퀵 뷰
- Zone B, C에는 필터 영향 없음 (전체 통합 노출)

### **Zone B: 심사 관리 + 피드백**
```
┌────────────────────────────┐
│ 심사 관리                   │
├────────────────────────────┤
│ 논문작성계획서              │
│ 심사 대기: 2건              │
│ → 심사하러 가기             │
├────────────────────────────┤
│ 본심사 (최종)               │
│ 심사 대기: 1건              │
│ → 심사하러 가기             │
└────────────────────────────┘

┌────────────────────────────┐
│ 피드백 (첨삭)               │
├────────────────────────────┤
│ 미답변 피드백               │
│ 대기: 1건 (3일 경과)        │
│ → 답변하러 가기             │
└────────────────────────────┘
```

**기능:**
- 단계별 미심사 건수 표시
- 미답변 피드백 강조 (빨간 배경)
- 클릭 시 해당 화면으로 이동

### **Zone C: 학사 일정 + 미팅**
```
┌────────────────────────────┐
│ 주요 학사 일정              │
├────────────────────────────┤
│ [D-3] 예비심사 결과 입력 마감│
│ 11.25 ~ 11.29 본심사 기간   │
└────────────────────────────┘

┌────────────────────────────┐
│ 나의 지도 미팅              │
├────────────────────────────┤
│ [승인 대기] 2건             │
│ - [박사] 김철수 (11.28)     │
│   [온라인]                  │
│   → 승인하러 가기           │
├────────────────────────────┤
│ [예정된 미팅] 2건           │
│ - [온라인] 오늘 14:00       │
│   이영희 (Zoom)             │
└────────────────────────────┘
```

**기능:**
- D-Day 계산 및 표시
- 승인 대기 / 예정된 미팅 분리
- 온라인/오프라인 구분

## 🔧 주요 기능

### 1. **사이드바 토글**
```javascript
function toggleSidebar() {
    const sidebar = document.getElementById('sidebar');
    sidebar.classList.toggle('collapsed');
}
```

### 2. **학위 과정 필터**
```javascript
function changeDegreeFilter(filterId) {
    currentDegreeFilter = filterId;
    renderDegreeFilter();
    renderProcessMap();
}
```

### 3. **퀵 뷰 패널**
```javascript
// PC: 마우스 호버
onmouseenter="showQuickView('proposal')"
onmouseleave="hideQuickView()"

// Mobile: 클릭
onclick="showQuickView('proposal')"
```

### 4. **네비게이션**
- `navigateToStudent(studentId)` - 학생 상세 화면
- `navigateToReview(tab)` - 심사관리 화면
- `navigateToFeedback()` - 온라인피드백 화면
- `approveMeeting(meetingId)` - 미팅 승인 화면

## 📱 반응형 디자인

### 데스크톱 (1024px 이상)
- 2열 그리드 레이아웃
- 사이드바 항상 표시
- 프로세스 맵 가로 배치

### 태블릿 (640px ~ 1024px)
- 1열 그리드 레이아웃
- 사이드바 숨김 (햄버거 메뉴)
- 프로세스 맵 세로 배치

### 모바일 (640px 이하)
- 필터 탭 줄바꿈
- 폰트 크기 축소
- 터치 최적화

## 🚀 실행 방법

1. `index.html` 파일을 브라우저에서 열기
2. 또는 로컬 서버 실행:
   ```bash
   # Python
   python -m http.server 8000

   # Node.js
   npx http-server
   ```
3. 브라우저에서 `http://localhost:8000` 접속
