## ⚡ 5분 테스트 가이드

### **1단계: 피드백 목록 확인 (30초)**
```
1. 메인 화면 → [제출물 피드백] 클릭
2. 목록에서 표절률 표시 확인
   ✅ "카피킬러: 12% / GPT킬러: 5%"
   ✅ "결과보고서 보기" 링크
3. 아무 제출물 클릭 → PDF 뷰어 열림
```

### **2단계: 판서 테스트 (1분)** 🔥
```
1. [판서] 도구 선택
2. PDF에 낙서 (선 그리기)
3. 5페이지로 이동 (스크롤 또는 입력)
4. 코멘트 패널에서 [1페이지로 이동] 클릭
5. ✅ 판서 정상 표시 확인
6. 다른 페이지로 이동
7. ✅ 판서가 다른 페이지에 안 나타남 확인
```

### **3단계: 영역 지정 + 마커 (1분)** 🔥
```
1. [영역지정] 도구 선택
2. PDF에서 드래그로 영역 지정
3. ✅ 마커(①) 즉시 표시 확인
4. 다른 영역 지정
5. ✅ 마커(②) 즉시 표시 확인
6. F12 → Console에서 "Added annotation" 로그 확인
```

### **4단계: 텍스트 선택 + 마커 위치 (1분)** 🔥
```
1. [선택] 도구 클릭
2. PDF 텍스트 드래그로 선택
3. ✅ 마커가 텍스트 위에 정확히 표시
4. ✅ 페이지 밖으로 안 나감
5. 여러 텍스트 선택 반복
```

### **5단계: 코멘트 연결 (1분)** 🔥
```
1. 코멘트 ① [연결] 버튼 클릭
2. ✅ 파란 테두리 확인
3. 코멘트 ② [연결] 버튼 클릭
4. ✅ 연결 완료, 테두리 해제
5. ✅ 두 카드에 "연결된 코멘트: 💬 2" 표시
```

### **6단계: UI 확인 (1분)**
```
1. 모든 버튼에 레이블 확인
   ✅ [등록] [음성] [자주쓰는 코멘트] [첨부] [연결]

2. 자주 쓰는 코멘트 팝오버
   ✅ 화면 아래에서도 위로 표시
   ✅ [✏️] [🗑️] 버튼 확인

3. 패널 너비
   ✅ 제출 이력: 250px
   ✅ 코멘트 패널: 450px
   ✅ 입력창 충분한 높이

4. 외곽선
   ✅ 본문 내용 잘 보임
```

## 🐛 문제 발생 시

### **판서가 안 보여요**
```
1. F12 → Console 탭
2. 에러 메시지 확인
3. fabricCanvas.getObjects() 입력
4. 객체 목록에 Path가 있는지 확인
```

### **마커가 안 보여요**
```
1. F12 → Console 탭
2. "Added annotation" 로그 확인
3. "Marker N: left=X, top=Y" 로그 확인
4. #marker-container 확인 (개발자 도구)
```

### **마커 위치가 이상해요**
```
1. Console에서 left/top 값 확인
2. 음수거나 너무 큰 값이면 버그
3. Group 객체인지 확인 (type=group)
```
