## 🎨 핵심 기능

### 1. 평가표 시스템

**평가표 종류 (관리자가 등록)**:
```javascript
1. 연구계획서 평가표
   - 연구 주제의 적절성 (20%)
   - 연구 방법의 타당성 (25%)
   - 선행연구 검토의 충실성 (20%)
   - 연구 계획의 구체성 (20%)
   - 형식 및 표현의 적절성 (15%)
   - 합격 기준: 70점

2. 중간논문 평가표
   - 연구 진행의 충실성 (30%)
   - 연구 방법의 적절성 (25%)
   - 결과 분석의 타당성 (25%)
   - 논문 작성의 완성도 (20%)
   - 합격 기준: 75점

3. 최종논문 평가표
   - 연구의 독창성 (25%)
   - 연구 방법의 타당성 (20%)
   - 결과 및 고찰의 적절성 (25%)
   - 논문의 완성도 (20%)
   - 학술적 기여도 (10%)
   - 합격 기준: 80점
```

### 2. 점수 계산 로직

```javascript
// 가중 평균 계산
총점 = Σ (항목별 점수 / 최대 점수 × 가중치)

// 예시
- 항목1: 8/10 × 20% = 16점
- 항목2: 7/10 × 25% = 17.5점
- 항목3: 9/10 × 20% = 18점
- 항목4: 7/10 × 20% = 14점
- 항목5: 8/10 × 15% = 12점
────────────────────────────
총점: 77.5점 → ✅ 합격 (기준: 70점)
```

### 3. 합불 판정 프로세스

```
[심사위원 평가 제출]
   ↓
[모든 심사위원 제출 완료]
   ↓
[시스템 자동 판정]
   - 평균 점수 계산
   - 합격 기준과 비교
   ↓
[심사위원장 검토]
   - 모든 평가 조회
   - 점수 비교
   - 총평 확인
   ↓
[최종 결정]
   ├─ 승인: 시스템 판정 확정
   ├─ 보류: 재평가 요청
   └─ 반려: 판정 변경
   ↓
[학생/지도교수 알림]
```

## 🧪 테스트 시나리오

### **시나리오 1: 심사위원 평가 (P003 - 김교수)**

```
1. index.html 열기
2. [심사관리] 메뉴 클릭
3. 목록 확인:
   - RA001: 김철수 - 연구계획서 (🔹 심사위원, 진행중)
   - RA002: 이영희 - 중간논문 (⭐ 심사위원장, 심사완료)
   - RA003: 박민수 - 최종논문 (🔹 심사위원, 승인)

4. "김철수 - 연구계획서" 클릭
5. 평가표 작성:
   - 연구 주제의 적절성: 8점
   - 연구 방법의 타당성: 7점
   - 선행연구 검토: 9점
   - 연구 계획: 7점
   - 형식 및 표현: 8점
   → 실시간 총점: 76점 (✅ 합격)

6. 총평 작성:
   - 장점: 연구 주제 선정 우수
   - 개선점: 연구 방법 구체화 필요
   - 종합: 전반적으로 우수

7. [💾 임시저장] 클릭 → 토스트 "저장되었습니다"
8. [✅ 제출] 클릭 → 확인 모달 → 제출 완료
9. 제출된 평가 화면으로 전환
```

### **시나리오 2: 심사위원장 최종 결정 (P003 - 김교수)**

```
1. [심사관리] 메뉴
2. "이영희 - 중간논문" 클릭 (⭐ 심사위원장)
3. 내 평가 확인 (제출 완료)
4. 하단으로 스크롤 → "⭐ 종합 평가" 섹션

5. 평가 현황 확인:
   ✅ 김교수 (위원장) - 11/15 14:30 제출
   ✅ 이교수 (위원) - 11/16 16:20 제출
   ✅ 한교수 (위원) - 11/14 18:45 제출

6. 점수 비교표:
   항목                    김교수  이교수  한교수  평균
   연구 진행의 충실성      9      8      9      8.7
   연구 방법의 적절성      8      9      8      8.3
   결과 분석의 타당성      8      7      8      7.7
   논문 작성의 완성도      9      8      9      8.7
   총점                   85     80     85     83.3

7. 시스템 판정:
   - 평균: 83.3점
   - 기준: 75점 이상
   - 판정: ✅ 합격

8. 최종 결정:
   - [승인] 선택
   - 의견: "모든 평가가 우수. 최종논문 진행 승인"
   - [✅ 최종 결정 제출]

9. 토스트: "최종 결정이 제출되었습니다"
10. 목록으로 돌아가기 → 상태: 🟢 승인
```

### **시나리오 3: 필터링 및 검색**

```
1. [심사관리] 메뉴
2. 필터 테스트:
   - 단계: [연구계획서] → 1건 표시
   - 역할: [심사위원장] → 해당 건만 표시
   - 상태: [승인] → 승인된 건만 표시
   - [전체] 복원

3. 검색 테스트:
   - "김철수" 입력 → 해당 학생 심사 표시
   - "블록체인" 입력 → 제목 포함 심사 표시
   - "2024" 입력 → 학번 포함 심사 표시
```
